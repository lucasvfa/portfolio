Dicas:
  • Comprima vídeos com HandBrake (grátis) — H.264, bitrate 2–4 Mbps
  • Comprima imagens com TinyPNG ou Squoosh (web)
  • Use JPG para fotos, PNG para gráficos com transparência, WebP para ambos
  • Vídeos de cover devem ter entre 3–8 segundos, sem áudio
  • Evite GIFs — use MP4, é 10x mais leve com mesma qualidade
  • Nome dos arquivos: sem espaços, sem acentos (ex: meu_projeto_1.jpg)
  • Total do site ideal: < 50MB para carregamento rápido

Histórico de Versões

📚 Como funciona o seu site

▼
① EDITAR — Faça suas mudanças aqui no dashboard
  • Troque textos, fotos, vídeos, ordem dos projetos
  • Tudo que você muda fica salvo automaticamente neste navegador
② SALVAR VERSÃO — Clique em "Salvar Versão" (botão no topo)
  • Isso cria um "checkpoint" — como um save point de videogame
  • Se algo der errado, você pode voltar pra essa versão
  • Dica: salve uma versão ANTES de fazer mudanças grandes
③ GERAR SITE — Clique em "Publicar" (botão verde no topo)
  • Isso gera um arquivo .zip com o site completo (HTML + imagens + vídeos)
  • Você pode escolher o nome do arquivo
  • Tudo já vem organizado dentro do .zip — não precisa montar nada
④ PUBLICAR — Suba os arquivos no Cloudflare Pages
  • Descompacte o .zip baixado
  • Abra dash.cloudflare.com no navegador
  • Vá em "Workers & Pages" no menu lateral
  • Clique no seu projeto (lucasvaz ou similar)
  • Clique em "Create deployment"
  • Arraste a pasta resultante para o Cloudflare Pages
  • Aguarde o upload — em 1-2 minutos seu site está no ar!

📄 Guia de cada aba

▼
🎨 Aba "Projeto"

O que é: Aqui você edita o conteúdo de cada projeto individual

Como navegar: Use os botões com nome dos projetos (as "bolhinhas" coloridas) OU o dropdown no topo

Informações do projeto: Clique na barra com o nome pra abrir/fechar os campos

  • Nome: O nome que aparece no site
  • Tipo: Categorias do projeto (ex: "Art Direction", "Motion Design"). Digite e aperte Enter pra adicionar
  • Ano: Quando o projeto foi feito
  • Cliente: Nome do cliente
  • Descrição: Texto curto sobre o projeto

Cover Image: A imagem grande que aparece quando alguém abre o projeto (formato paisagem 16:9)

Thumbnail: A imagem pequena que aparece na homepage do site (o card do projeto)

Mídias do projeto: As imagens e vídeos que aparecem dentro do projeto

  • Clique em "Adicionar Mídia" pra subir novos arquivos
  • Ou arraste arquivos direto pra área de edição
  • Clique numa mídia pra selecionar e mudar o tamanho
  • Arraste o ícone ⣿ pra mudar a ordem

Tamanhos de mídia (quanto espaço ocupa na tela):

  • Inteiro: ocupa toda a largura
  • Metade: ocupa metade da tela (2 cabem lado a lado)
  • Terço: ocupa um terço (3 cabem lado a lado)
  • Largo: ocupa 2/3 da tela
  • Estreito: ocupa 1/3 da tela
🏠 Aba "Homepage Grid"

O que é: Aqui você organiza como os projetos aparecem na página inicial

Como usar:

  • Cada card é um projeto — a imagem do card é o Thumbnail
  • Arraste o ícone ⣿ pra mudar a ordem dos projetos
  • Clique num card e use os botões de tamanho (Inteiro, Metade, Terço, Largo, Estreito)
  • Duplo-clique num card pra ir editar o projeto
  • O preview à direita mostra como vai ficar no site real

Os números no canto inferior esquerdo são a ordem de aparição. O símbolo no canto inferior direito mostra o tamanho atual.

⚙ Aba "Config. do Site"

O que é: Textos gerais do site, redes sociais, informações de contato

  • Seção EN (azul): Textos em inglês — aparece quando o visitante escolhe English
  • Seção PT (verde): Textos em português — aparece quando o visitante escolhe Português

Campos explicados:

  • Hero Bio (frase curta): A primeira frase que aparece no topo do site
  • Hero Bio (frases poéticas): O texto maior logo abaixo, com formatação
  • Sobre/About: O texto da seção "Sobre" com sua história
  • Label/Título da seção pessoal: Nome da seção de trabalhos pessoais
  • Contato: Textos da seção de contato
  • Footer: Texto do rodapé (ex: "© 2026 Lucas Vaz")

Barra de formatação (B I S A ¶ ↵):

  • B = Negrito (texto mais grosso)
  • I = Itálico (texto inclinado)
  • S = Destaque forte (similar ao negrito, mas com significado de ênfase)
  • A = Cor de destaque (texto fica na cor rosa/accent do site)
  • ¶ = Novo parágrafo (pula uma linha e começa bloco novo)
  • ↵ = Quebra de linha (vai pra linha de baixo sem pular linha)
  • 🔗 = Inserir link (transforma texto em link clicável)
  • Como usar: Selecione o texto que quer formatar, depois clique no botão
  • O botão "Preview" mostra como o texto vai ficar no site real

Geral: Email de contato e skills/habilidades

Redes Sociais: Adicione seus perfis (Instagram, Behance, LinkedIn, etc.)

📅 Aba "Histórico"

O que é: Todas as versões que você salvou

  • "Salvar Versão Atual": Cria um novo checkpoint com a data e hora

Cada versão mostra: nome, data/hora, e botões:

  • Restaurar: Volta o dashboard pra essa versão (CUIDADO: substitui o que tem agora)
  • Baixar: Baixa um .zip completo com tudo: dados da versão, site (index.html), dashboard (admin-9975.html) e todos os arquivos de mídia. É um backup completo que funciona em qualquer computador.
  • Deletar: Remove essa versão do histórico

Importar Versão: Carrega um arquivo .json que você ou alguém salvou antes

Dica: Sempre salve uma versão antes de fazer mudanças grandes!

❓ Perguntas Frequentes

▼
Salvei as mudanças mas o site não mudou?
"Salvar Versão" guarda suas mudanças aqui no dashboard, mas NÃO publica no site. Pra atualizar o site, clique em "Publicar" e depois faça o upload no Cloudflare Pages.
Fiz algo errado, como volto atrás?
Vá na aba "Histórico" e clique em "Restaurar" na versão anterior. Por isso é importante salvar versões antes de fazer mudanças grandes.
Posso usar esse dashboard em outro computador?
Suas mudanças ficam salvas no navegador deste computador. Pra levar pro outro: vá em Histórico, clique em "Baixar" na versão mais recente — vai baixar um .zip completo com tudo (site, dashboard, imagens, dados). No outro computador, descompacte o zip, abra o admin-9975.html, e use "Importar Versão" com o arquivo versao.json ou o próprio .zip.
O que acontece se eu limpar o histórico do navegador?
Você perde TODAS as mudanças não publicadas. Por isso: 1) Salve versões regularmente, 2) Baixe backups .json, 3) Gere o site depois de cada sessão de edição.
Que tamanho devem ter as imagens?
Cover/Hero: 1920x1080 (formato de TV), máx 500KB. Thumbnail: mín 800x600, máx 300KB. Imagens do projeto: 1200-1920px de largura, máx 500KB. Use TinyPNG.com pra comprimir.
Como comprimo vídeos?
Baixe o HandBrake (handbrake.fr, grátis). Abra o vídeo, escolha formato MP4, codec H.264, bitrate 2-4 Mbps. Ideal: vídeos entre 3-5MB.
Posso adicionar um projeto novo?
Clique em "+ Novo Projeto" no topo. Dê um nome, preencha as informações, adicione cover, thumbnail e mídias.
Como mudo a ordem dos projetos na homepage?
Vá na aba "Homepage Grid" e arraste os cards pela alça ⣿ na posição que quiser.
O que é o botão ⋯ no canto?
É um menu com opções extras: resetar layouts, exportar dados, trocar senha e ver o status do armazenamento.
Meu vídeo não aparece no site
Verifique se é MP4 (formato H.264). Outros formatos como MOV ou AVI não funcionam em navegadores. Converta com o HandBrake.
